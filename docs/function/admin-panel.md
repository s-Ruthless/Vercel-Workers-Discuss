# 后台设置

管理后台用于审核评论、删除评论、管理说说和系统配置。

访问地址：`https://your-project.vercel.app/admin/`

## 多站点管理

管理后台顶部导航栏提供了站点下拉选择器，用于在多站点之间切换数据视图。

在「站点管理」页面可以添加、编辑、删除托管站点，每个站点通过 `siteId` 进行数据隔离。

## 设置

### 头像服务前缀

常用的 Gravatar 镜像服务：

| 服务 | 前缀地址 |
| --- | --- |
| Gravatar 官方 | `https://gravatar.com/avatar` |
| Cravatar (国内) | `https://cravatar.cn/avatar` |
| 自定义镜像 | `https://your-mirror.com/avatar` |

### 显示博主标签

开启是否显示博主标签，配置博主邮箱，即可在评论中显示博主标签；关闭为不显示。

标签文字可选填，留空则使用默认图标。

### 评论置顶

在评论管理页面，点击"编辑"按钮可修改评论的置顶权重：

- `1`：不置顶（默认值）
- `2` 或更高：数值越大，该评论在列表中排序越靠前

置顶后的评论会在前台评论列表中优先显示。

### 评论审核

开启「新评论审核后再显示」后，普通用户提交的新评论默认状态为"待审核"，需在后台审核通过后才会显示；关闭时，新评论将直接公开展示。

## 使用官方管理后台

使用官方提供的管理后台（最新版本）：<https://vwd.zishu.me>

登陆时填入你的 API 地址、账号和密码。

## 评论数据修改

支持单条评论的修改，包括状态、内容等。

::: warning 注意
如若批量修改评论信息，请前往 Vercel Postgres 控制台执行 SQL 语句。

高危操作，请保证你熟悉 SQL 语句，避免对评论数据造成不可逆的损失。建议使用前先导出评论数据备份。
:::

## 自部署管理后台

```bash
cd admin
# 安装依赖
npm install
# 开发环境启动（默认端口 5173）
npm run dev
# 生产环境构建
npm run build
# 本地预览生产构建结果
npm run preview
```

构建产物在 `public/admin/` 目录，随项目一起部署到 Vercel。

管理后台基于 Vite + Vue 3 + TypeScript 的单页应用，用于管理评论和系统配置。
